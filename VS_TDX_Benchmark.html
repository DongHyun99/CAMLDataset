VS-TDX Benchmark

KAIST
Data Hugging Face Paper

VS-TDX evaluates Vision-Language Models on sensor-specific reasoning across thermal, depth, and X-ray imagery.

Overview

The VS-TDX benchmark is designed to evaluate whether VLMs deeply understand sensor-specific visual information. It targets the gap between ordinary image understanding and reasoning over the physical properties of thermal, depth, and X-ray sensors.

The benchmark contains approximately 10K question-answer pairs grouped into multi-vision perception and multi-vision understanding tasks.

Dataset Information

Key Application

vision-language model evaluation, sensor-specific reasoning, thermal/depth/X-ray understanding

Primary Data Type

Images, Questions, Answers, Evaluation code

Dataset Characteristics

  • Three sensor types: Thermal, Depth, and X-ray
  • Six question types: Existence, Count, Position, Scene Description, Contextual Understanding, and Sensor Understanding
  • Approximately 10K question-answer pairs

Labels

Questions are evaluated using answer options such as A, B, C, D, yes, and no.
