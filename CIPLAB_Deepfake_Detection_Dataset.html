CIPLAB Deepfake Detection Dataset

Yonsei University

CIPLAB Deepfake Detection Dataset combines FF++-derived fake data and DFDC cropped-face benchmarks for robust deepfake detection and capability analysis.

Overview

This dataset collection supports deepfake training, evaluation, and analysis. It includes self-blending pseudo-fakes from FF++ real videos, high-quality SimSwap fake samples, cropped DFDC face frames, and video-level analytics from a Gated Swin-DINO model.

Dataset Information

Key Application

deepfake detection, robustness evaluation, capability analysis, uncertainty analysis

Primary Data Type

Images, Face-cropped frames, Video labels, Analytics metadata, Landmarks

Dataset Characteristics

  • FF++ Self-Blending Pseudo-Fake: 21,817 images
  • FF++ SimSwap High-Quality Fake: 21,817 images
  • DFDC Cropped-Face Benchmark: 178,370 images
  • DFDC Capability-Metrics Analytics: 5,000 videos

Labels

DFDC video labels use 0 for real and 1 for fake. Analytics metadata includes prediction probability, entropy, gate weights, and frequency summary statistics.
