Jeju Autonomous Driving Dataset (v1)

RideFlux

A real-world dataset collected by RideFlux during autonomous vehicle pilot operations in Jeju Island.

Overview

Autonomous Driving Data Collection

  • Collection of data from various angles, including the side-rear and side-front views of autonomous vehicles.
  • Collection of abnormal and adverse weather data (e.g., foggy conditions).

    • Autonomous Driving Data Object Labeling

  • Bounding box annotations for the location/size and attributes (e.g., occlusion, signal status) of objects (vehicles, motorcycles, pedestrians, traffic lights, etc.)
  • Lane marking annotations (e.g., dashed lines, solid lines, color).

    • Dataset Information

    Key Application

    Autonomous Driving, Car/Vehicle Perception, Situation Awareness, Drivability Assessment, etc

    Dataset Characteristics

    Volume of Datset

  • Bounding Box: 54,000 images with 490,000 labeled objects.

    • Labels

    Data format

  • Labeled positions for vehicles, pedestrians, traffic lights, and lanes (vehicles, pedestrians, traffic lights: bounding boxes; lanes: polylines).
  • For vehicle labels, additional attributes such as occlusion and truncation are annotated.
  • Vehicle Label Classes: Two-wheeled vehicles, trucks, buses, others (passenger cars), unknown (unidentifiable).
  • For lane markings, annotations include type (solid line, dashed line, stop line) and color (white, yellow, others).
    • Each annotation records the type corresponding to a unique ID, and the annotations are stored in the annotationsData field in the form of bounding boxes or polygons based on this ID.

    Directory Structure

    Each folder contains metadata indicating the image capture date and the specific camera on the autonomous vehicle that recorded the footage. Within each folder, there are img and json subfolders, where the json files share the same names as the corresponding images. These JSON files store information essential for autonomous driving, such as data on people, pedestrians, traffic lights, and lane markings.
