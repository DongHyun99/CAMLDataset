IHP Dataset

ETRI
Data

The Irregular Human Posture (IHP) dataset is a proposed training dataset designed to recognize various human postures captured in real-world surveillance cameras.

Overview

To address real-world human postures, we constructed a dataset comprising six classes: ‘Walking’, ‘Crouching’, ‘Lying’, ‘Standing’, ‘Running’, and ‘Sitting’. Additionally, the dataset is annotated with segmentation maps and keypoints to provide a more detailed understanding of human postures.

To ensure robustness across various scenarios, we also included images containing relatively small instances to account for varying object sizes.

Dataset Information

Key Application

Smart Surveillance System

Primary Data Type

Training images, Test images, Train annotations.json, Test annotations.json

Data Function

Training, testing

Dataset Characteristics

  • Total number of images: 41,904
  • Total number of classes: 6
  • Average image resolution: 720p
