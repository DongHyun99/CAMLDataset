Foggy KITTI Dataset

ETRI
Data

Foggy images at various intensities.

Overview

This repository shares the Foggy KITTI dataset introduced and utilized in the ECCV 2024 paper, MonoWAD: Weather-Adaptive Diffusion Model for Robust Monocular 3D Object Detection, Youngmin Oh, Hyung-Il Kim, Seong Tae Kim†, Jung Uk Kim†.

The Foggy KITTI dataset extends the KITTI dataset to include challenging weather conditions, aiming to support research in real-world applications such as autonomous driving. It contains synthetic fog images with different levels of intensity and is divided into training and testing sets, providing a useful resource for developing and evaluating models in practical scenarios.

Dataset Information

Key Application

Monocular 3D Object Detection, Adverse Weather Removal, Scene Understanding, etc

Primary Data Type

RGB Images, Annotations

Data Function

Training, testing

Dataset Characteristics

  • Total number of images: 7,481 training images (3,712 training and 3,769 validation), 7,518 testing images
  • Adverse weather levels: 0.05, 0.1, 0.15, 0.3 (0.1 used as the main)
  • Average image resolution: 1242 x 375

    • Labels

    Each image is annotated with bounding boxes, object class labels
