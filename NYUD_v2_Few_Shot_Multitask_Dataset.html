NYUD-v2 Few-Shot Multi-Task Dataset

Kyung Hee University

A compact NYUD-v2 few-shot multi-task dataset with task-aware sample selection for semantic segmentation, depth estimation, surface normal estimation, and object boundary detection.

Overview

This dataset selects 40 samples, approximately 5% of the original 795 NYUD-v2 training samples, and uses them as a compact training set with 654 evaluation samples.

Instead of using a single random subset for every task, the dataset applies task-specific sampling criteria so that each few-shot training split reflects the characteristics required by the corresponding task.

Dataset Information

Key Application

few-shot multi-task learning, semantic segmentation, depth estimation, surface normal estimation, object boundary detection, indoor scene understanding

Primary Data Type

RGB images, semantic labels, depth maps, surface normals, object boundaries

Data Function

Few-shot training and evaluation for NYUD-v2 based multi-task learning.

Dataset Characteristics

  • 40 training samples selected from 795 original NYUD-v2 training samples.
  • 654 samples are used for evaluation.
  • The training subset corresponds to approximately 5% of the original NYUD-v2 training split.
  • Each task uses a sampling criterion aligned with the task characteristics.

Task-Specific Sampling

  • Semantic Segmentation: 40 samples are selected so that every class appears at least once and all classes are used during training.
  • Depth Estimation: The training samples are divided into 40 bins according to maximum depth, and one sample is randomly selected from each bin to cover diverse distance ranges.
  • Surface Normal Estimation: 40 samples are randomly selected from the full training dataset because diverse geometric orientations naturally appear across indoor scenes.
  • Object Boundary Detection: 40 samples are randomly selected from the full dataset because object boundaries are generally distributed across most samples, reducing selection bias.
