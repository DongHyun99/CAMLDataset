GYM/DMC-CDIL Dataset

KAIST

Gym/DMC-CDIL Dataset is the sets of demonstrations collected for training policy models via Cross-Domain Imitation Learning (CDIL) on Gym or DeepMind Control Suite (DMC). The dataset was introduced in our paper "Domain Adaptive Imitation Learning with Visual Observation" (D3IL).

Overview

Gym/DMC-CDIL Dataset are collected from original Gym or DMC environments, or from their modified versions. It is intended to be used for CDIL research for the purposes permitted under our Data License.

Each set of demonstrations includes 10,000 image observations. Each observation consists of 4 RGB images, which is the 4 consecutive timesteps ($o_t, o_{t-1}, o_{t-2}, o_{t-3}$) of rendered images. The resolution of each RGB image ($o_t$) varies from 32x32, 48x48, or 64x64.

In the D3IL paper, three sets of demonstrations are used for each CDIL task, and each CDIL task consists of a source domain and a target domain. For example, if the source domain is "Reacher2-v2" and the target domain is "Reacher3-v2", then we use "Reacher2-v2" expert data (Source-Expert; SE), "Reacher2-v2" non-expert data (Source-NonExpert; SN), and "Reacher3-v2" non-expert data (Target-NonExpert; TN).

Dataset Information

Key Application

Imitation Learning, Cross-Domain Imitation Learning, Reinforcement Learning

Primary Data Type

Image Observations, Indices

Data Function

Expert, Prior

Dataset Characteristics

  • Total number of images: 220,000
  • Total number of images per set: 10,000
  • Total number of expert sets: 8
  • Total number of non-expert sets: 14
  • Image resolution: 32 x 32, 48 x 48, or 64 x 64 pixels (depending on the environments)

    • Labels

    There are no labels in the dataset.
