PersonaDrive Dataset

Kyung Hee University

PersonaDrive pairs real-world NavSim/OpenScene driving scenes with natural-language passenger intents and persona-specific ego trajectories.

Overview

PersonaDrive is a large-scale persona-conditioned driving dataset for controllable and instruction-following trajectory planning research. Each scene is annotated for nine driver personas spanning urgency and comfort preferences.

Every persona trajectory is paired with a natural-language passenger utterance that expresses the desired driving style.

Dataset Information

Key Application

instruction-following planning, controllable trajectory generation, driver/passenger preference modeling, language-conditioned autonomous driving

Primary Data Type

Scene JSON, Trajectories, Natural-language intents, Persona labels

Dataset Characteristics

  • 115,434 scene files, one JSON per NavSim/OpenScene scene token
  • Approximately 1,038,906 labeled trajectories across nine personas
  • Approximately 1,038,906 natural-language passenger utterances
  • Raw size is approximately 1.4 GB as uncompressed JSON

Labels

The nine persona categories are UH_CL, UH_CM, UH_CH, UM_CL, UM_CM, UM_CH, UL_CL, UL_CM, and UL_CH.
