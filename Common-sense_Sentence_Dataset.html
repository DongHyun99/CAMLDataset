Common-sense Sentence Dataset

KAIST

The sentence dataset we built is a collection of textual descriptions reflecting common-sense knowledge about various classes. The resulting dataset provides a rich source of semantic knowledge, enabling its integration into tasks like Open Set Recognition (OSR) to improve model reasoning and performance against novel or unseen classes.

Overview

The sentence dataset constructed provides common-sense knowledge for OSR tasks. Sentences were extracted for each class in CIFAR10 and TinyImageNet using varying minimum weight thresholds (0, 0.2, 0.5, 1.0, 1.2, 1.5). Higher thresholds result in fewer, higher-quality sentences, while lower thresholds yield a larger quantity of lower-quality sentences. This trade-off ensures the dataset is versatile, balancing quality and quantity to adapt to the needs of different datasets and recognition tasks.
